Filename: ComptageDeMots.py
Il faut se mettre dans le dossier où est le programme avant de l'exécuter.

Ce programme prend comme input un ensemble de TextGrids, et sort en output une feuille csv (par TextGrid) qui énumère tous les tours de parole, et pour chacun indique le nombre de mots.)

Avant d'exécuter le programme comptageDeMots, mettre les TextGrids dont on veut compter les mots dans le répertoire ./Textgrids_a_Compter/

La liste des fichiers à compter figure dans le fichier d'output <date>ListeEnquetesACompterCSV
